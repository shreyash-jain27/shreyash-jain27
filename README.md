## 🏆 Featured Projects

### 🔹 [HookFlow](https://github.com/shreyash-jain27/HookFlow) — Webhook Delivery Engine
Reliable webhook delivery system with BullMQ/Redis queuing, HMAC signature verification, idempotency guarantees, and Dead Letter Queue handling for failed deliveries.
`Node.js` `Redis` `BullMQ` `Docker`

### 🔹 [DataWhisper](https://github.com/shreyash-jain27/DataWhisper) — Natural Language to SQL
Converts plain-English queries into SQL using Google Gemini / Ollama, enabling non-technical users to query databases conversationally.
`Python` `LLM` `SQL`

### 🔹 [DEVFLOW](https://github.com/shreyash-jain27/DEVFLOW) — AI-Enhanced Task Management
Task management backend with JWT authentication, real-time updates via Socket.io, MongoDB persistence, and AI-powered task assistance using the Claude API.
`Node.js` `MongoDB` `Socket.io` `JWT` `Claude API`

### 🔹 [HomeFixAI](https://github.com/shreyash-jain27/HomeFixAI) — Home Services Frontend
Frontend application for booking home repair services, built with a modern, responsive UI.
`React` `TypeScript` `Tailwind CSS`
